# Decisions

## 2026-05-19 - 第一条课程包方向

- 第一条课程包目标：从前端背景出发，补齐 TypeScript 与后端能力，最终独立完成一个“学习系统”主题的全栈项目。
- 能力目标：可以胜任初级全栈工程师，而不只是做出一个演示项目。
- 学习者背景：10 年前端经验，能独立做 React 页面，但 TypeScript 薄弱，后端零基础。
- 时间投入：工作日每天 2-3 小时，周六日每天约 8 小时。
- Codex 协作策略：早期优先提示和检查，让学习者亲手做；必要时切到平衡协助，但必须记录独立程度。
- 关键风险：后端思维建立困难、缺少正向成果反馈导致难以坚持、Codex 辅助造成虚假掌握。
- 设计要求：每个 session 尽量产生可见成果、可运行产物、明确反馈或小胜利，同时用证据和 rubric 防止“感觉会了”。
